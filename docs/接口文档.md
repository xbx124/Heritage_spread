# 接口文档

> 非遗文化数字化传播系统 · 后端 API 说明  
> **本文档由组长定义，成员B按此实现，成员C/D按此调用，定好后不随意改动**

## 基础约定

- 基础地址：`http://localhost:8080/api`
- 数据格式：所有请求和响应均为 JSON
- 字段命名：统一使用**下划线**（如 `project_name`，不用 `projectName`）
- 统一响应结构：

```json
{
  "code": 200,
  "msg": "success",
  "data": { }
}
```

| code | 含义 |
|------|------|
| 200  | 成功 |
| 400  | 参数错误 |
| 404  | 数据不存在 |
| 500  | 服务器错误 |

## 一、全局搜索

### 关键词搜索（C调用 · 搜索框用）

```
GET /api/search?keyword=昆曲
```

**说明**：在项目名称、地区、传承人姓名中模糊匹配，后端需 JOIN 多张表。

**请求参数**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| keyword | string | 是 | 搜索关键词 |

**返回示例**

```json
{
  "code": 200,
  "msg": "success",
  "data": [
    {
      "type": "project",
      "id": 2,
      "name": "昆曲",
      "area": "江苏苏州",
      "description": "中国现存最古老的戏曲剧种之一"
    },
    {
      "type": "inheritor",
      "id": 2,
      "name": "汪世瑜",
      "project_name": "昆曲"
    }
  ]
}
```

## 二、非遗项目

### 获取项目列表（C调用 · 首页展示用）

```
GET /api/projects
```

**请求参数（可选）**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| category_id | int | 否 | 按分类筛选 |
| area | string | 否 | 按地区筛选 |
| page | int | 否 | 页码，默认 1 |
| size | int | 否 | 每页条数，默认 10 |

**返回示例**

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "total": 4,
    "list": [
      {
        "project_id": 1,
        "project_name": "景德镇手工制瓷技艺",
        "category_name": "传统技艺",
        "area": "江西景德镇",
        "create_time": "2025-05-01 10:00:00"
      }
    ]
  }
}
```

### 获取项目详情（C调用 · 详情页用）

```
GET /api/projects/:id
```

**返回示例**

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "project_id": 1,
    "project_name": "景德镇手工制瓷技艺",
    "category_name": "传统技艺",
    "area": "江西景德镇",
    "project_intro": "千年瓷都景德镇独有的手工拉坯、绘瓷技艺...",
    "create_time": "2025-05-01 10:00:00",
    "inheritors": [
      {
        "inheritor_id": 1,
        "name": "王龙根",
        "years": 40,
        "intro": "景德镇手工拉坯第四代传承人..."
      }
    ]
  }
}
```
