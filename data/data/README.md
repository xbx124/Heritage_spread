# 统计分析模块（成员D负责）

## 功能概述

本模块包含传播数据可视化图表页面，调用后端统计接口，使用 ECharts 展示非遗文化传播数据的统计分析。

## 文件结构

```
data/
├── index.html          # 统计分析主页面
├── js/
│   └── stats.js        # 数据统计与图表渲染逻辑
└── README.md           # 模块说明文档
```

## 页面功能

### 1. 数据汇总卡片
- 总浏览量统计
- 总曝光量统计
- 传播渠道数量
- 非遗项目数量

### 2. 可视化图表
| 图表名称 | 类型 | 数据源 | 说明 |
|---------|------|--------|------|
| 各渠道传播数据对比 | 柱状图 | `/api/stats/channel` | 展示各渠道的浏览量和曝光量 |
| 各项目传播数据对比 | 柱状图 | `/api/stats/project` | 展示各非遗项目的浏览量和曝光量 |
| 浏览量渠道分布 | 环形饼图 | `/api/stats/channel` | 展示各渠道浏览量占比 |
| 项目曝光量排行 | 水平条形图 | `/api/stats/project` | 按曝光量排序展示项目 |
| 传播数据综合分析 | 组合图 | 双数据源 | 渠道与项目浏览量综合对比 |

## 技术实现

- **前端框架**: 原生 HTML/CSS/JavaScript
- **图表库**: ECharts 5.5.0
- **字体**: Noto Serif SC（思源宋体）、ZCOOL XiaoWei（站酷小薇体）
- **API 基础地址**: `http://localhost:8080/api`

## 接口调用

本模块调用以下后端统计接口：

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/stats/channel` | GET | 获取各渠道传播统计数据 |
| `/api/stats/project` | GET | 获取各项目传播统计数据 |

接口响应格式：
```json
{
  "code": 200,
  "msg": "success",
  "data": [
    { 
      "channel_name": "渠道名称", 
      "total_view": 10000, 
      "total_exposure": 30000 
    }
  ]
}
```

## 使用方式

1. 确保后端服务已启动（默认端口 8080）
2. 直接打开 `index.html` 即可访问统计页面
3. 页面会自动加载并展示传播数据可视化图表

## 特性

- 响应式布局，适配不同屏幕尺寸
- 数据实时刷新功能
- 优雅的视觉设计，融合中国传统元素配色
- 图表支持交互（悬停提示、图例切换等）
